BILLING RATES
================================================================================

Tiered Pricing Structure:
- Tier 1 (0-100 kWh): M1.20 per kWh
- Tier 2 (101-300 kWh): M1.50 per kWh
- Tier 3 (Above 300 kWh): M2.00 per kWh

Calculation Example (150 kWh):
- First 100 kWh: 100 × M1.20 = M120.00
- Remaining 50 kWh: 50 × M1.50 = M75.00
- Total Bill: M120.00 + M75.00 = M195.00

Wait, let me recalculate based on the actual bill:
For 150 kWh usage resulting in M245.00:
- First 100 kWh: 100 × M1.20 = M120.00
- Remaining 50 kWh: 50 × M1.50 = M75.00
- Additional charges may apply

Calculation Example (350 kWh):
- First 100 kWh: 100 × M1.20 = M120.00
- Next 200 kWh: 200 × M1.50 = M300.00
- Remaining 50 kWh: 50 × M2.00 = M100.00
- Total Bill: M120.00 + M300.00 + M100.00 = M520.00 ✓

================================================================================
TESTING INSTRUCTIONS
================================================================================

1. START APPLICATION
   - Run the application using: ./mvnw javafx:run
   - The login screen will appear

2. LOGIN
   - Use either admin/admin123 or staff/staff123
   - Click "Sign In" button

3. DASHBOARD
   - After login, you'll see the main dashboard
   - Options: Manage Customers, Calculate Bills, View Reports, Logout

4. MANAGE CUSTOMERS
   - View existing customers (C001, C002)
   - Add new customers
   - Edit existing customer information
   - Delete customers
   - Calculate bills for selected customers
   - Search customers by ID or name

5. CUSTOMER OPERATIONS
   - ADD: Click "Add Customer" and fill in all fields
   - EDIT: Select a customer, click "Edit Customer"
   - DELETE: Select a customer, click "Delete Customer"
   - CALCULATE: Select a customer, click "Calculate Bill"

6. BILL CALCULATION
   - Select a customer from the table
   - Click "Calculate Bill"
   - Enter electricity usage in kWh
   - Click "Calculate Bill" to see the amount
   - Click "Save Bill" to update customer record
