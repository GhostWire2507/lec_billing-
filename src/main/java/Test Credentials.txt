BILLING CALCULATION FORMULA
------------------------------
The system uses tiered pricing:

- 0-100 kWh: M1.20 per kWh
- 101-300 kWh: M1.50 per kWh
- Above 300 kWh: M2.00 per kWh

Example: 350 kWh usage
- First 100 kWh: 100 Ã— M1.20 = M120.00
- Next 200 kWh: 200 Ã— M1.50 = M300.00
- Remaining 50 kWh: 50 Ã— M2.00 = M100.00
- TOTAL: M520.00
